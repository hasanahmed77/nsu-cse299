# Cloud-Based Movie Streaming Platform

A cloud-based movie streaming platform inspired by Netflix, developed as a full-stack project for **CSE299 - Junior Design** at North South University.

---

## Overview

This project aims to build a **personal cloud-based streaming platform** where legally obtained movies and series can be hosted, browsed, and streamed. The platform provides:

- User registration and login
- Movie browsing and search
- Watchlist management
- Admin panel for uploading and managing content

It simulates the architecture of modern streaming platforms, giving hands-on experience with **full-stack web development**.

---

## Features

### User Features

- Register, login, and logout
- Browse movies and series
- Search for movies or series
- Add movies/series to watchlist
- View movie/series details (release date, IMDB rating, cast)

### Admin Features

- Add and manage movies or series
- Upload content to cloud storage
- Desktop-first responsive interface
